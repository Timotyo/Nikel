ControlCenter4를 사용하여 문서를 PDF 파일로 스캔

0. ControlCenter4 모드 화면이 나타나면 고급 모드를 선택하고 확인을 클릭합니다.

1. 파일을 클릭합니다.

2. 스캔을 누릅니다.
스캔 프로세스가 시작됩니다. 스캔한 데이터가 저장되는 폴더가 자동으로 열립니다.

  1. 파일 형식 풀다운 목록에서 PDF(*.pdf)를 선택합니다.
  2. 문서에 사용할 파일 이름을 입력할 수 있습니다.
  3. 파일을 기본 폴더에 저장하거나, (찾아보기) 버튼을 클릭하여 원하는 폴더를 선택할 수 있습니다.
  4. 해상도 풀다운 목록에서 스캔 해상도를 선택할 수 있습니다.
  5. 문서 크기 풀다운 목록에서 문서 크기를 선택할 수 있습니다.
