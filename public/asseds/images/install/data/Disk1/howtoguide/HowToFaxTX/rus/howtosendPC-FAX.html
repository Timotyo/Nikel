Отправка факсов через программу PC-FAX

Отправка файла с компьютера

1. Создайте файл в любом приложении на компьютере.
2. В меню Файл выберите пункт Печать.
Открывается диалоговое окно Печать:

3. Выберите принтер Brother PC-FAX и нажмите кнопку Печать. Открывается диалоговое окно передачи факса PC-FAX:

4. Введите номер факса и нажмите кнопку Старт.

Отправка сканированного изображения с помощью ControlCenter4

0. Если отображается экран "Режим ControlCenter4", выберите Расширенный режим, а затем нажмите кнопку OK.

1. Выберите вкладку PC-FAX.
2. Нажмите кнопку (Отправить отсканированное изображение), чтобы начать сканирование.

3. Открывается диалоговое окно передачи факса PC-FAX.

4. Введите номер факса и нажмите кнопку Старт.
Примечание
Чтобы изменить настройки сканирования, щелкните правой кнопкой мыши кнопку (Отправить отсканированное изображение), а затем измените настройки.
