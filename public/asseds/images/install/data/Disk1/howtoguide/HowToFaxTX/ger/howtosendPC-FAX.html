PC-Fax senden

Eine Datei vom Computer aus senden

1. Erstellen Sie auf Ihrem PC mit einer beliebigen Anwendung eine Datei.
2. Klicken Sie auf Datei, dann auf Drucken.
Das Dialogfeld Drucken erscheint:

3. Wählen Sie Brother PC-FAX als Drucker aus und klicken Sie dann auf Drucken. Das Dialogfeld
PC-FAX senden wird angezeigt:

4. Geben Sie eine Faxnummer ein und klicken Sie auf Start.

Eingescanntes Bild mit ControlCenter4 versenden

0. Wenn der Bildschirm ControlCenter4-Modus angezeigt wird, wählen Sie Erweiterter Modus und klicken Sie dann auf OK.

1. Wählen Sie die Registerkarte PC-FAX.
2. Klicken Sie auf (Gescanntes Bild senden), um den Scanvorgang zu starten.

3. Das Dialogfeld PC-FAX senden wird angezeigt.

4. Geben Sie eine Faxnummer ein und klicken Sie auf Start.
Hinweis
Wenn Sie die Scaneinstellungen ändern möchten, klicken Sie mit der rechten Maustaste auf (Gescanntes Bild senden) und nehmen Sie die Änderungen vor.
