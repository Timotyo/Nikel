パソコンからファイルを送る

1. アプリケーションでファイルを作成する。
2. アプリケーションの[ファイル]メニューから[印刷]を選択する。
下記のような印刷ダイアログが表示される。

3. プリンターの選択で[Brother PC-FAX]を選択し、印刷を押す。下記のようなPC-FAX送信ダイアログが表示される。

4. 送り先のファクス番号を入力し、[送信]をクリックする。

ControlCenter4でスキャンして送る

0. 下記のような画面が表示されたら、Advancedモードを選択します。

1. [PC-FAX]タブを選択する。
2. [画像ファイルを送信]を選択する。
スキャンが開始します。

3. 下記のようなPC-FAX送信画面が表示される。

4. 送り先のファクス番号を入力し、[送信]をクリックする。
補足
[画像ファイルを送信]のアイコンを右クリックすることで、スキャンの設定を変更することができます。
