PC-FAX 发送

从计算机发送文件

1. 在计算机的任一应用程序内创建一个文件。
2. 点击文件，然后点击打印。
将出现打印对话框：

3. 选择 Brother PC-FAX 作为打印机，然后点击打印。 出现 PC-FAX 发送对话框：

4. 输入传真号码，然后点击开始。

使用 ControlCenter4 发送已扫描图像

0. 若显示 ControlCenter4 模式屏幕，请选择高级模式，然后点击确定。

1. 选择 PC-FAX 选项卡。
2. 点击 (发送扫描的图像) 开始扫描。

3. 出现 PC-FAX 发送对话框。

4. 输入传真号码，然后点击开始。
提示
若要更改扫描设置，请右击 (发送扫描的图像)，然后更改设置。
