PC-FAX 傳送

從電腦傳送檔案

1. 在您電腦上的任何應用程式中建立一個檔案。
2. 按一下檔案，再按一下列印。
列印對話方塊將會顯示：

3. 選擇 Brother PC-FAX 作為印表機，再按一下列印。 PC-FAX 傳送對話方塊將會顯示：

4. 輸入傳真號碼，再按一下開始。

使用 ControlCenter4 傳送掃描影像

0. 若顯示 ControlCenter4 模式畫面，請選擇進階模式，再按一下確定。

1. 選擇 PC-FAX 標籤。
2. 按一下 (傳送掃描影像) 開始掃描。

3. PC-FAX 傳送對話方塊將會顯示。

4. 輸入傳真號碼，再按一下開始。
註釋
如果您想變更掃描設定，請在 (傳送掃描影像) 上按一下滑鼠右鍵，再變更設定。
