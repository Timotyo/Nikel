Réception PC-FAX

Affichage des nouveaux messages PC-FAX
  1. Cliquez sur Recevoir.
  2. Suivez les instructions affichées à l'écran.
  3. L'icône PC-FAX s'affiche dans la barre d'état système de l'ordinateur.
  4. Double-cliquez sur l'icône PC-FAX.
Remarque
Pour rétablir le mode de réception de télécopie par défaut, mettez la fonction de réception PC-FAX sur OFF depuis les paramètres de télécopie en appuyant sur Menu ou sur l'appareil.
Les télécopies reçues sur l'ordinateur sont enregistrées dans le dossier suivant.
Windows XP : C:\Documents and Settings\All Users\Documents\BrFaxRx\PC-FAX Rx
Windows Vista ou ultérieur : C:\Users\Public\Documents\BrFaxRx\PC-FAX Rx
