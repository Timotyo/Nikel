Ricezione PC-FAX

Visualizzazione dei nuovi messaggi PC-FAX
  1. Fare clic su Ricezione.
  2. Seguire le istruzioni visualizzate sullo schermo.
  3. Sulla barra delle applicazioni del computer viene visualizzata l'icona PC-FAX.
  4. Fare doppio clic sull'icona PC-FAX.
Nota
Per ripristinare sull'apparecchio la modalità di ricezione fax predefinita, impostare la Ricezione PC-FAX su NO dalle impostazioni fax, premendo Menu oppure sull'apparecchio.
I fax ricevuti sul computer vengono salvati nella cartella indicata di seguito.
Windows XP; C:\Documents and Settings\All Users\Documents\BrFaxRx\PC-FAX Rx
Windows Vista o versioni successive; C:\Users\Public\Documents\BrFaxRx\PC-FAX Rx
