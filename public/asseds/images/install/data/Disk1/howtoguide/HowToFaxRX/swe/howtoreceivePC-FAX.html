PC-FAX-mottagning

Visa nya PC-FAX-meddelanden
  1. Klicka på Ta emot.
  2. Följ anvisningarna på skärmen.
  3. PC-FAX-ikonen visas i datorns aktivitetsfält.
  4. Dubbelklicka på PC-FAX-ikonen.
OBS
Om du vill återgå till standardläget för faxmottagning ska du ställa in PC-FAX-mottagning på Av i faxinställningarna genom att trycka på Menu eller på din maskin.
Faxmeddelanden som tas emot på din dator sparas i nedanstående mapp.
Windows XP: C:\Documents and Settings\All Users\Documents\BrFaxRx\PC-FAX Rx
Windows Vista eller senare: C:\Users\Public\Documents\BrFaxRx\PC-FAX Rx
