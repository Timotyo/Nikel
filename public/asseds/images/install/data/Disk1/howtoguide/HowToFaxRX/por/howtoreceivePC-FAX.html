Recepção PC-FAX

Visualizar novas mensagens de PC-FAX
  1. Clique em Receber.
  2. Siga as instruções que aparecem no ecrã.
  3. O ícone PC-FAX aparece no tabuleiro de tarefas do computador.
  4. Clique duas vezes no ícone PC-FAX.
Nota
Se pretender que o equipamento volte ao modo de recepção de fax predefinido, altere a Recepção PC-FAX para Desl. nas definições de Fax premindo Menu ou no equipamento.
Os faxes recebidos no seu computador são guardados na pasta que se segue.
Windows XP: C:\Documents and Settings\All Users\Documents\BrFaxRx\PC-FAX Rx
Windows Vista ou posterior: C:\Users\Public\Documents\BrFaxRx\PC-FAX Rx
