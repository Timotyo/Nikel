Прием факсов через программу PC-FAX

Просмотр новых сообщений PC-FAX
  1. Нажмите Получить.
  2. Следуйте инструкциям, отображаемым на экране.
  3. На панели задач компьютера появится значок PC-FAX.
  4. Дважды щелкните значок PC-FAX.
Примечание
Чтобы переключить устройство обратно в режим приема факсов по умолчанию, нажмите Меню или на устройстве и в пункте "Настройки факса" выберите для параметра "Прием PC-FAX" значение "ВЫКЛ".
Факсы, принимаемые на компьютере, будут сохраняться в следующей папке.
Для Windows XP: C:\Documents and Settings\All Users\Documents\BrFaxRx\PC-FAX Rx
Для Windows Vista или более поздней версии: C:\Users\Public\Documents\BrFaxRx\
