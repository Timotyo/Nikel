PC-FAX 수신중

새 PC-FAX 메시지 보기
  1. 수신을 클릭합니다.
  2. 화면의 지침을 수행합니다.
  3. PC-FAX 아이콘이 컴퓨터의 작업 트레이에 나타납니다.
  4. PC-FAX 아이콘을 더블 클릭합니다.
참고
시스템을 팩스 수신 모드 초기 설정으로 되돌리려면, 시스템에서 메뉴(Menu) 또는 을 누르고 팩스 설정에서 PC-FAX 수신을 OFF로 변경합니다.
수신한 팩스는 컴퓨터의 다음 폴더에 저장됩니다.
Windows XP; C:\Documents and Settings\All Users\Documents\BrFaxRx\PC-FAX Rx
Windows Vista 이상; C:\Users\Public\Documents\BrFaxRx\PC-FAX Rx
