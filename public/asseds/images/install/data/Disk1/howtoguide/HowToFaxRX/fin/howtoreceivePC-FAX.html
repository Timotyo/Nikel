PC-FAX-vastaanotto

Uusien PC-FAX-viestien tarkastelu
  1. Napsauta Vastaanota.
  2. Noudata näytön ohjeita.
  3. PC-FAX-kuvake tulee näkyviin tietokoneen ilmaisinalueelle.
  4. Kaksoisnapsauta PC-FAX-kuvaketta.
Vinkki
Jos haluat palauttaa laitteen oletusasetuksena olevaan faksinvastaanottotilaan, muuta faksiasetuksissa PC-FAX-vastaanotto EI-asetukseen painamallaMenu tai laitteessa.
Tietokoneen vastaanottamat faksit tallennetaan seuraavaan kansioon.
Windows XP: C:\Documents and Settings\All Users\Documents\BrFaxRx\PC-FAX Rx
Windows Vista tai uudempi: C:\Users\Public\Documents\BrFaxRx\PC-FAX Rx
