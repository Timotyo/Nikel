パソコンでファクスを見ます
  1. [PC-FAX 受信を起動]をクリックする。
  2. パソコンに表示される画面に従ってください。
  3. PC-FAX受信アイコンがパソコンのタスクトレイに表示される。
  4. PC-FAX受信アイコンをダブルクリックする。
補足
本製品でファクスを印刷するときは、本製品の[メニュー]、または[ ]のファクス設定から
[PCファクス受信]をオフしてください。
パソコンで受信したファクスは下記フォルダに保存されます。
Windows XP： C:\Documents and Settings\All Users\Documents\BrFaxRx\PC-FAX Rx
Windows Vista以降： C:\Users\Public\Documents\BrFaxRx\PC-FAX Rx
