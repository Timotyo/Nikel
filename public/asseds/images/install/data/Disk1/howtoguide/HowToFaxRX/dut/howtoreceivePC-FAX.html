PC-FAX ontvangen

Nieuwe PC-FAX-berichten weergeven
  1. Klik op Ontvangen.
  2. Volg de instructies op het scherm.
  3. Het pictogram PC-FAX wordt weergegeven op de taakvak van het computerscherm.
  4. Dubbelklik op het pictogram PC-FAX.
Opmerking
Als u uw machine weer in de standaardmodus voor het ontvangen van faxen wilt zetten, zet u PC-FAX ontv. bij de faxinstellingen op UIT door op Menu of op de machine te drukken.
Faxen die op uw computer worden ontvangen, worden opgeslagen in de volgende map.
C:\Documents and Settings\All Users\Documents\BrFaxRx\PC-FAX Rx (bij gebruik van Windows XP)
C:\Users\Public\Documents\BrFaxRx\PC-FAX Rx (bij gebruik van Windows Vista of recenter)
