PC-FAX empfangen

Neu empfangene PC-Faxe ansehen
  1. Klicken Sie auf Empfangen.
  2. Folgen Sie den Anweisungen auf dem Bildschirm.
  3. Das PC-FAX-Symbol erscheint nun in der Taskleiste.
  4. Doppelklicken Sie auf das PC-FAX-Symbol.
Hinweis
Wenn Sie das Gerät auf den Standard-Faxempfang zurücksetzen möchten, ändern Sie die Einstellung für PC-FAX-Empfang in den Faxeinstellungen auf AUS, indem Sie am Gerät Menü (Menu) oder drücken.
Im Computer empfangene Faxe werden im folgenden Ordner gespeichert.
Windows XP; C:\Documents and Settings\All Users\Documents\BrFaxRx\PC-FAX Rx
Windows Vista oder höher; C:\Users\Public\Documents\BrFaxRx\PC-FAX Rx
