PC-FAX 接收

查看新的 PC-FAX 信息
  1. 点击接收。
  2. 遵循屏幕提示进行操作。
  3. PC-FAX 图标将出现在计算机任务栏中。
  4. 双击 PC-FAX 图标。
提示
如果您想将设备返回到默认传真接收模式，请按设备上的功能键或 ，然后在传真设置中将 PC-FAX 接收设置为关。
接收到计算机中的传真将保存在以下文件夹中。
Windows XP：C:\Documents and Settings\All Users\Documents\BrFaxRx\PC-FAX Rx
Windows Vista 或更高版本：C:\Users\Public\Documents\BrFaxRx\PC-FAX Rx
