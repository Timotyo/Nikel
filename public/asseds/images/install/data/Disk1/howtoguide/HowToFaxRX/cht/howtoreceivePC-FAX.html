PC-FAX 接收

查看新的 PC-FAX 訊息
  1. 按一下接收。
  2. 遵循螢幕上的說明執行操作。
  3. PC-FAX 圖示將會顯示在您的電腦工作欄中。
  4. 按兩下 PC-FAX 圖示。
註釋
如果您想將機器恢復到預設傳真接收模式，請按機器上的功能或 ，從傳真設定將 PC-FAX 接收設定為關。
接收到電腦的傳真將儲存在以下資料夾中。
Windows XP：C:\Documents and Settings\All Users\Documents\BrFaxRx\PC-FAX Rx
Windows Vista 或更高版本：C:\Users\Public\Documents\BrFaxRx\PC-FAX Rx
