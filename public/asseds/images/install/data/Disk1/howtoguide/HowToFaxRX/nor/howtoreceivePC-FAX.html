PC-FAKS-mottak

Vise nye PC-FAKS-meldinger
  1. Klikk Motta.
  2. Følg instruksjonene på skjermen.
  3. PC-FAKS-ikonet vises i datamaskinens oppgavelinje.
  4. Dobbeltklikk PC-FAKS-ikonet.
Merk
Hvis du vil at maskinen skal gå tilbake til standard faksmottakmodus, kan du endre PC-FAKS-mottak til AV fra faksinnstillingene ved å trykke Menu eller på maskinen.
Fakser som mottas på datamaskinen, blir lagret i den følgende mappen.
Windows XP: C:\Documents and Settings\All Users\Documents\BrFaxRx\PC-FAX Rx
Windows Vista eller senere: C:\Users\Public\Documents\BrFaxRx\PC-FAX Rx
